# Spring Cloud Netflix Eureka 

## Introduction
Service Discovery是基于微服务的体系结构的关键原则之一。尝试手动配置每个客户端或某种形式的约定可能很难做到并且可能很脆弱。Eureka是Netflix服务发现服务器和客户端。可以配置和部署服务器以使其具有高可用性，每个服务器将注册服务的状态复制到其他服务器

## Development Environment

- Spring Boot : 2.1.7
- gradle : 5.6
- Java : jdk1.8
